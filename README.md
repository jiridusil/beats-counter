# Beats Counter

A simple web app for counting exercise beats across rounds with configurable pauses.

## Features

- Define beats per round, number of rounds, and BPM for each exercise
- 6 built-in exercise presets with individually saved settings (localStorage)
- Auto-chain presets — start the first one, the rest run automatically
- Configurable pause between exercises with ambient sound and 3-2-1 countdown
- English / Czech language toggle
- No dependencies — plain HTML, CSS, and JavaScript

## Deploy

```bash
firebase use <project-id>
firebase deploy
```
